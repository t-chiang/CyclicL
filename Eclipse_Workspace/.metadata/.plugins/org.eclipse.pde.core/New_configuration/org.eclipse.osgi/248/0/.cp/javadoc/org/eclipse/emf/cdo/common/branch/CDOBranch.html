    public interface CDOBranch
extends CDOBranchPoint, CDONameProvider, org.eclipse.net4j.util.container.IContainer<CDOBranch>, java.lang.Comparable<CDOBranch>, IAdaptable
    Represents a stream of changes that is isolated from other streams of changes.

    A branch starts at a fixed base point and ends at a floating head point. Between these two points there can be a number of other branch points:

    • Commit infos are points in a branch that represent commit operations.
    • Branch tags are named points in a branch.
    • Base points of sub branches of a branch.
